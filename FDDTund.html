FDD (What We Did in Class)

The class took place on 13.05.2025.
The lesson topic was FDD (Feature Driven Development).
In class, the teacher talked about FDD and its different stages.
The teacher explained that FDD is an Agile methodology focused on feature development.
The stages of FDD were:

  • Develop an overall model
  • Build a features list
  • Plan by feature
  • Design by feature
  • Build by feature
To practice these stages, we did various exercises in Miro.
We did stages 1 - 4 in Miro with the projects: ToDo list and Spotify app.
